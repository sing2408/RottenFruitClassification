Dataset used : https://www.kaggle.com/datasets/sriramr/fruits-fresh-and-rotten-for-classification
This is a code/program that can distinguish or classify whether a given fruit picture is rotten or fresh. The models used were split into 3 categories : transfer learning, fine tuning and fusion model (proposed model). pre-trained models used : MobileNetV2, Xception, Inception, VGG16, InceptionV3
